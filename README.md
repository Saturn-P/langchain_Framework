# langchain


- simple chains and prompts
- question-answering over documents
- using memory and tools (planned)
- guidance for running examples locally

Requirements:
- Python 3.8+
- see `requirements.txt`

Getting started:
1. Clone the repo
2. Create and activate a virtualenv
3. pip install -r requirements.txt
4. Set your OpenAI (or other provider) API key in the environment, e.g. `export OPENAI_API_KEY=...`
5. Run `python examples/simple_chain.py`
